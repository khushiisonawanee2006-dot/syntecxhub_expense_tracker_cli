# Expense_Tracker_CLI
A Python-based command-line application to manage income and expenses.  
This project allows users to add transactions, view monthly summaries, export reports, and generate expense charts.

---

## Features

- Add income and expense entries
- Store records in CSV file
- View all transactions
- Generate monthly financial summary
- Export reports to CSV
- Generate expense charts using Matplotlib

---

## Technologies Used

- Python
- CSV Module
- Datetime Module
- Matplotlib
